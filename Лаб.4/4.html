Селекторы CSS

Изначально изменили цвет прямоугольника №2(#d2),соответственно,он не поменял свой цвет, хотя и относится к классу first

Селектор определяет, к какому элементу применять то или иное CSS-правило. 

	*{ 
	font-family: Arial, Verdana, sans-serif; /* Рубленый шрифт для текста */ 
	font-size: 96%; /* Размер текста */ 
	}

Псевдокласс — это селектор, который выбирает элементы, находящиеся в специфическом состоянии, например, они являются первым элементом своего типа, или на них наведён указатель мыши. Они обычно действуют так, как если бы вы применили класс к какой-то части вашего документа, что часто помогает сократить избыточные классы в разметке и даёт более гибкий, удобный в поддержке код 

INPUT:focus { 
    color: red; /* Красный цвет текста */
   }

Псевдоэлементы ведут себя сходным образом, однако они действуют так, как если бы вы добавили в разметку целый новый HTML-элемент, а не применили класс к существующим элементам. Псевдоэлементы начинаются с двойного двоеточия :: 

.new::after {
    content: " - Новьё!"; /* Добавляем после текста абзаца */
    color: red; /* Красный цвет текста */
    font-style: italic; /* Курсивное начертание */
   }


CSS селекторы
Селектор Пример Описание
#id #d1 Выбирает элемент с идентификатором d1 (id="d1")
.class .second Выбирает все элементы с классом second (class="second")
* * Выбор всех элементов
[Атрибут] [title] Выбирает все элементы с атрибутом title
[Атрибут*="значение"] [title*="один"] Выбирает все элементы с атрибутом title, в значении которого (в любом месте) встречается подстрока (в виде отдельного слова или его части) "один" (title="один и два")
[Атрибут$="значение"] [src$=".png"] Выбирает все элементы с атрибутом src, значение которого оканчивается на ".png" (src="some_img.png")
Элемент span Выбор всех элементов span
